# 04. 평가 기준 수정하기

공고 문구나 평가지표가 바뀌었을 때만 하면 됩니다.

## 파일

**하나만** 수정합니다.

```
automations/recruitment_slack/config/hiring_rubric_2026.md
```

포지션별로 `## 서비스기획자`, `## CFO` 같은 헤더로 구역이 나뉘어 있습니다. 해당 섹션만 편집하고 UTF-8로 저장하세요.

## Cursor에 반영

다음부터 `PROMPT_ROUNDHR.md`로 요청하면 **자동으로 새 내용**이 프롬프트에 포함됩니다. (`@config/hiring_rubric_2026.md` 참조)

## `job_id`와 섹션 매칭

라운드HR의 `job_id`와 공고 제목이 바뀌면, Cursor 프롬프트에서 “`job_id=6001`은 서비스 기획자 섹션”처럼 한 줄 힌트를 추가해 두면 판정이 안정적입니다. 필요 시 `PROMPT_ROUNDHR.md` 안 지시문을 고치면 됩니다.

## 팁

- 필수 자격·탈락 조건을 눈에 띄게 써 두면 C등급 판정이 일관됩니다.
- 서비스기획자처럼 **점수표(S/A/B 컷)**가 있는 포지션은 그대로 두고, Cursor가 등급을 맞추도록 프롬프트에 “내부 표의 S/A/B/C 정의를 따를 것”이라고 적어 두었습니다.
