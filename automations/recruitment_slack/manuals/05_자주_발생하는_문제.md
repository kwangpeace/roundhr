# 05. 자주 발생하는 문제

검은 창(터미널)에 뜬 오류 문구를 보고 아래 표에서 찾으시면 됩니다.

## 빈번한 오류 7가지

### 1) `not_in_channel`

- **의미**: Slack 봇이 해당 채널에 **초대되어 있지 않음**.
- **해결**: Slack에서 해당 채널 열고  
  `/invite @봇이름`  
  후 `post_message.bat` 다시 실행.

### 2) `channel_not_found`

- **의미**: `.env` 의 `SLACK_CHANNEL_ID` 가 **잘못된 값** 이거나, 봇이 접근 불가한 비공개 채널.
- **해결**:  
  1. Slack에서 해당 채널 열기 → 상단 채널 이름 클릭 → 하단 "채널 ID" 복사  
  2. `.env` 의 `SLACK_CHANNEL_ID` 교체, 저장  
  3. 다시 실행

### 3) `invalid_auth` 또는 `token_revoked`

- **의미**: Slack 봇 토큰이 만료/폐기됨.
- **해결**: Slack 앱 설정에서 **새 Bot Token** 재발급 → `.env` 의 `SLACK_BOT_TOKEN` 교체.

### 4) `message.md 가 없습니다`

- **의미**: 글자 그대로. 파일을 만들지 않았거나 이름이 다름(예: `message.txt`).
- **해결**: 파일명은 **정확히 `message.md`** 로. 확장자 숨김 옵션을 켜두면 `.md` 가 안 보일 수 있으니, 탐색기 **"파일 확장명" 보기** 를 켜고 확인하세요.

### 5) `ModuleNotFoundError: No module named 'xxx'`

- **의미**: 파이썬 패키지가 설치되지 않음.
- **해결**: 터미널에서 해당 폴더로 이동해  
  ```
  .\.venv\Scripts\python -m pip install -r requirements.txt
  ```  
  실행 후 다시 시도.

### 6) `python ... 은(는) ... 인식되지 않습니다`

- **의미**: Python이 PC에 설치되지 않음.
- **해결**: [python.org](https://www.python.org/downloads/) 에서 3.12 이상 설치. 설치 시 **"Add python.exe to PATH"** 체크.

### 7) Slack에 메시지가 안 보여요 (오류는 없는데)

- **확인 포인트**  
  1. 창에 `[done] Slack 전송 성공.` 이 떴는지  
  2. **다른 Slack 워크스페이스**를 보고 있진 않은지  
  3. `.env` 의 채널 ID가 보고 있는 채널과 같은지  
  4. 봇이 그 채널에 초대되어 있는지

## 터미널 창의 한글이 깨져 보여요

- **Slack에 도착한 메시지는 정상** 입니다.
- Windows 기본 콘솔이 CP949라서 그렇게 보이는 것뿐입니다.
- 필요하면 `post_message.bat` 실행 후 `chcp 65001` 같은 명령으로 UTF-8로 바꿀 수 있지만 **운영에는 영향 없음**.

## 그래도 해결 안 되면

1. 오류 **창을 그대로 스크린샷** 하거나, 맨 아래 몇 줄을 복사  
2. 담당자에게 공유 → 대개 원인이 금방 잡힙니다.
