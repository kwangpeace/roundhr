# 05. 자주 발생하는 문제

## `not_in_channel`

- 봇이 그 채널에 초대되지 않음.
- Slack에서 `/invite @봇이름` 후 다시 전송.

## `channel_not_found`

- 채널 ID가 틀렸거나, 봇이 접근 못 하는 채널.
- 채널 정보에서 ID를 다시 복사해 `토큰|채널` 의 채널 부분만 교체.

## `invalid_auth` / `token_revoked`

- 봇 토큰 만료·폐기.
- [api.slack.com/apps](https://api.slack.com/apps)에서 토큰 재발급 후, 실행 시 **새 토큰**을 다시 붙여넣기 (파일에 저장했다면 그 파일만 교체).

## `message.md 가 없습니다`

- `automations/recruitment_slack/message.md` 파일명·위치 확인. 확장자 `.md` 인지 탐색기에서 “파일 확장명” 표시 켜고 확인.

## `인자 형식: "xoxb-...|C..."`

- `토큰`과 `채널ID` 사이에 **파이프(`|`) 하나**만 있고 공백이 없어야 합니다.
- 전체를 큰따옴표로 감싸서 실행: `send_slack.py "xoxb-abc|C0123"`

## `ModuleNotFoundError`

```powershell
.\.venv\Scripts\python -m pip install -r requirements.txt
```

## Python이 인식되지 않음

- [python.org](https://www.python.org/downloads/)에서 3.12+ 설치, PATH 옵션 체크.

## 전송 성공인데 Slack에 안 보임

- 다른 워크스페이스/채널을 보고 있지 않은지, 채널 ID가 맞는지 확인.

## 한글이 터미널에서 깨져 보임

- Slack 메시지는 보통 정상입니다. Windows 콘솔 인코딩 때문일 수 있습니다.

## 그래도 안 되면

오류 메시지 **마지막 몇 줄**을 복사해 담당자에게 전달하세요.
