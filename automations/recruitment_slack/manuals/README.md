## 핵심 파일 위치

- 운영 스크립트: `automations/recruitment_slack/`
  - `post_message.bat` – **Slack 전송 버튼** (매일 누르는 파일)
  - `message.md` – **Cursor에서 정리한 결과를 붙여넣는 파일** (없으면 새로 만들기)
  - `message.template.md` – 메시지 양식 참고용
  - `config/jd/*.txt` – 공고별 평가 기준
  - `.env` – 토큰·키 (외부에 노출 금지)
