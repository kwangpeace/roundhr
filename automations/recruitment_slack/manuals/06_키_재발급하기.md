# 06. 키 재발급하기 (보안)

Slack 봇 토큰이 채팅·문서·Git 등에 노출됐다면 **즉시 재발급**하세요.

## Slack 봇 토큰

1. [https://api.slack.com/apps](https://api.slack.com/apps) → 해당 앱
2. **OAuth & Permissions** → **Regenerate** 또는 앱 재설치
3. 새 `xoxb-...` 복사
4. 앞으로 `send_slack.bat` / `send_slack.py` 실행 시 **새 토큰**을 `xoxb-...|C...` 형식으로만 입력

이 레포에는 토큰을 저장하지 않으므로, **`.env`를 고칠 필요는 없습니다.** (예전에 `.env`를 썼다면 그 파일을 삭제하거나 비워 두세요.)

## 재발급 후 확인

`message.md`에 테스트 문구 한 줄 넣고 `send_slack.bat`로 전송해 보면 됩니다.

## Git에 실수로 올린 경우

1. 토큰 재발급으로 기존 토큰 무효화
2. 레포에서 해당 커밋·파일 정리 후 푸시
3. 이력까지 지워야 하면 `git filter-repo` 등은 엔지니어에게 요청
