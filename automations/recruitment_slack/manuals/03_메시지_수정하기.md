# 03. 메시지 수정하기 (`message.md`)

Cursor 결과를 사람이 다듬거나, 민감 정보를 지우거나, 특정 지원자 블록을 빼고 싶을 때입니다.

## 파일 위치

`automations/recruitment_slack/message.md`

## 규칙

- Slack이 이해하는 **간단한 마크다운** 위주 (`*굵게*`, `_기울임_`, `` `코드` ``)
- 형식 예시는 `PROMPT_ROUNDHR.md` 안 Slack용 마크다운 블록과 같게 맞추면 읽기 좋습니다.

## 자주 하는 편집

- **한 명 삭제**: 해당 지원자 블록(등급·근거·리스크)만 지우고, 상단 인원 수 문구가 있으면 숫자 조정
- **등급만 변경**: `*S*` 등 알파벳만 바꾸고, 목록 순서(S→A→B→C) 정리
- **공고 블록 통째로 삭제**: 해당 공고 섹션과 `---` 구분선까지 제거

## 저장 후

1. `Ctrl+S` 저장
2. `send_slack.bat` 실행 후 `토큰|채널` 입력

## 실수로 다 지웠을 때

- Slack에 이미 보낸 메시지를 복사해 `message.md`에 다시 넣거나
- Cursor에 같은 프롬프트로 다시 요청합니다.

`message.template.md`는 제거되었습니다. 양식은 `PROMPT_ROUNDHR.md`를 기준으로 하세요.
