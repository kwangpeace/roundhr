# 01. 최초 세팅 (한 PC에 한 번만)

다른 노트북에서 처음 시작할 때만 진행하면 됩니다.

## 준비물

- Windows PC
- **Python 3.12 이상** ([python.org](https://www.python.org/downloads/) — "Add to PATH" 체크)
- **Cursor** + 라운드HR MCP ([공식 MCP 연동 가이드](https://guide.roundhr.com/round-ai/mcp) — 레포 요약은 [docs/ROUNDHR_MCP.md](../docs/ROUNDHR_MCP.md))
- Slack 워크스페이스에 봇 설치·채널 초대 완료

## 1) 폴더 열기

```
...\roundhr\automations\recruitment_slack
```

## 2) Slack 봇을 채널에 초대

```
/invite @봇이름
```

비공개 채널이면 이걸 안 하면 `not_in_channel` 오류가 납니다. 설정은 [docs/SLACK_SETUP.md](../docs/SLACK_SETUP.md).

## 3) Python 환경 (처음 한 번)

**방법 A** — `send_slack.bat` 더블클릭 후 프롬프트가 뜨면 아무 키나 눌러 닫기. 처음이면 `.venv` 생성·`requests` 설치가 자동으로 됩니다.

**방법 B** — PowerShell에서:

```powershell
cd automations\recruitment_slack
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
```

## 4) 동작 테스트

1. `message.md` 파일을 만들고 짧은 문장 한 줄을 넣어 저장합니다.
2. `send_slack.bat` 실행.
3. `Slack 인증 한 줄 입력:` 에 **`xoxb-...|C...`** 형식으로 붙여넣고 Enter.
4. Slack 채널에 문장이 보이면 성공.

> **토큰을 `.env`나 레포 파일에 넣지 않습니다.** 테스트 후에도 토큰은 메모장 등에만 두고 커밋하지 마세요.
