# S / A / B / C 등급 (서류 단계 보조)

상세 배점·필수 탈락 조건은 [`../config/hiring_rubric_2026.md`](../config/hiring_rubric_2026.md)의 **포지션별 섹션**을 따릅니다. 아래는 Slack 요약에 쓰는 공통 라벨 의미입니다.

| 등급 | 의미 | 서류 단계 권고 |
|------|------|----------------|
| **S** | 공고·지표 대비 매우 강함 | 즉시 후속 전형 권장 |
| **A** | 우수, 소폭 확인만 | 서류 통과 권장 |
| **B** | 조건부·추가 자료·면접에서 확인 | 보류·추가 검토 |
| **C** | 필수 미충족 또는 정합성 낮음 | 서류 통과 비권장 (최종은 담당자) |

모든 등급은 **보조 의견**이며 최종 판단은 채용 담당자에게 있습니다.
